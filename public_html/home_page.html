Coursework Details:
CW2 was to create a web front-end to the database created in CW1 using HTML CSS, PHP and JavaScript which will connect to my database and execute queries using PHP and SQL. The functionality we had to include was being able to add and delete and search for an actor and movie in the database
In order for the database to retain referential integrity an actor needs to be added before the movie, if the actor is not already in the table
